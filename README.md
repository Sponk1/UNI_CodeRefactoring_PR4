# UNI_CodeRefactoring_PR4

> Проект взят с GitHub для выполнения практической работы. 
> Ссылка на оригинал: [ссылка](https://github.com/no2ehi/simple-todo-app-react)

## Описание
Проект реализует небольшое web-приложение с базовым todo-list.
В приложении есть такой функционал:

- Добавление задач (название, дата, статус)
- Редактирование задач (название, дата, статус)
- Удаление задач
- Выполнение задачи (чекбокс)
- Фильтрация (выполнено, не выполнено, все)

## Аудит

### Проблемы

- Компоненты не придерживаются принципа SRP
- Плохая архитектура файловой системы
- Отсутсвие тестов

### Цель

Исправить вышеуказанные проблемы проекта, то есть изменить архитектуру проекта, разделить компоненты, провести тестирование.
